# 30 Jours de Gâteaux

Une application Android pour découvrir un gâteau différent chaque jour pendant 30 jours.  
Chaque gâteau contient un nom, une description et une image.

## Fonctionnalités

- Affichage d’une liste de gâteaux avec LazyColumn.
- Carte pour chaque gâteau avec animation d’expansion pour afficher la description.
- Chargement des images depuis Internet (via Coil).
- Thème avec fond rose bébé.
